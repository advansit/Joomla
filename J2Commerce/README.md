# J2Commerce Extensions

Professional [Joomla](https://github.com/joomla/joomla-cms) extensions for [J2Commerce](https://github.com/joomla-projects/j2commerce) e-commerce platform.

## Extensions

This repository contains the following extensions:

- **com_j2commerce_importexport** - Data import/export component
- **com_j2store_cleanup** - Migration cleanup tool
- **plg_j2commerce_acymailing** - AcyMailing integration plugin
- **plg_j2commerce_productcompare** - Product comparison plugin
- **plg_privacy_j2commerce** - GDPR compliance plugin
- **plg_system_j2commerce_2fa** - Two-Factor Authentication plugin

See individual extension folders for detailed documentation.
